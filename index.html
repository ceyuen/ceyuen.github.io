PlanCake

PlanCake was designed to be an all-in-one social event planning platform. Users can create events and invite their friends to help plan. Within an event, there can be multiple chat rooms for different issues such as flights or hotels where users can pin and like messages. If there's something that needs to be done, users can add it to a group task list. Lastly, finalized plans can be added to a detailed itinerary.
